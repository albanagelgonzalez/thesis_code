# Content of `analysis_code/`

The directory contains the following files for the analysis:

- `table.R/`: creation of table for further analysis with Nonpareil and MultiQC information
- `seq_check.R`: quality assessment of the data
- `alpha_div.R`: computation of alpha-diversity measures
- `beta_div_analysis.R`: evaluation of beta-diversity metrics
- `abundance_genus.ipynb`: Jupyter Notebook for producing abundance plots at the genus level
